## 오늘 한 것
- GitHub 가입하고 첫 레포지토리 생성
- Git 명령어 3개 익힘 (add, commit, push)
- add는 추가 뿐만이 아니라 상태 변경 느낌. 추가/변경/삭제 모두 add를 씀(git add 파일명)
- commit은 메모(git commit -m "내용")
- push는 add하고 commit한 파일을 git에 업데이트 하는 것(git push origin main)
