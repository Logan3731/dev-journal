# MyDay PRD

## 한 줄 정의
무엇을 해야할지 모르는 사람들이 스스로 질문하며 답을 낼 수 있도록 도와주는 앱

## 해결하는 문제
- 내가 해야하는 일들이 무엇인지 정리
- 왜 막막했는지 스스로 파악할 수 있음
- 기록을 확인하여 나의 역량 파악 가능

## 핵심 사용자
- 나
- 무언가 시작하지 못하는 사람들

## 필수 기능 (MVP)
1. 생각 정리
2. 우선순위 도출
3. 기록

## 의도적으로 빼는 기능
1. Ai api 연동 - ai가 답을 해주면 스스로 해결할 수 있는 역량 저하
